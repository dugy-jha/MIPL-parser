Image Controls

The Image Control plugin has various options to adjust the camera image:

Option to turn the image into a black and white image

Invert colours filter

Edge highlight filter

Histogram filter (a form of contrast optimization filter)

Noise reduction filter

Image sharpening filter

Image flip (flips the image along the horizontal axis)

Image mirror (mirrors the image along the vertical axis)

Brightness adjustment slider

Contrast adjustment slider
