How to run HP Diagnostics

If you think that there is hardware problem on HP computer you can run HP Diagnostics which will help you see where the potential problem is.

Step-by-step guide

Reboot computer

Press F2 in boot

Then choose diagnostics what you want to run

We recommend to run Fast System Test first, you can find this under System Tests

If that not find any problem you can run Component Test

If diagnostics found something error you should get failure id, please write that down and contact HP Support
