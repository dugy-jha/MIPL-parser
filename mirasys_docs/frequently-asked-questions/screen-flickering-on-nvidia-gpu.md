Screen flickering on Nvidia GPU
Problem

When you are using more than 2 screen it is possible that screens start flickering.

 This may happen when you are using Nvidia GTX -series.

Solution

You need to change Nvidia card settings in Nvidia Control Panel. In there change settings Prefer maximum performance in 3D setting options.
