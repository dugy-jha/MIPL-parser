Spotter show Direct3D error window
Problem

Spotter show Direct3D error window

Solution

When this error occured check that DirectX End-User Runtimes (June 2010) installed if not install it here https://www.microsoft.com/en-us/download/details.aspx?id=8109

If that not help update GPU drivers

Sometimes you need also reinstall Spotter software
