Correct order to stop and start VMS Services

If you are using an VMS installation example V9. You may need to stop VMS services when making changes on camera configuration files.

The correct order to stop VMS services is

WDServer

DVRServer

SMServer

And when you want to start them again, the correct order is

WDServer

DVRServer

SMServer
