Spotter Web layout saving fails

Some cases when there is manually installed Microsoft Internet Information Services (IIS) and not used installation script. There is possible that WebDAV component is installed part of this IIS. This can cause issues related to saving layouts or using PTZ controls.

On this case remove WebDAV component via Windows features.
