How to use Mirasys VMS camera motion detection

On default motion detection is done via Mirasys VMS. There is also possibility use camera own motion detection.

To use this

Check that manufacturer is supported

Check Supported Camera list

SD card for storing

On configuration side

First enable camera motion detection in camera settings

Configure motion detection levels in camera settings

Add camera to Mirasys VMS

Change in System Manager Cameras tab that this camera is using VMS and Camera hardware for motion detection

Set in scheduler side continuously recording

You can also set SD card to camera to store data there and if there is lost connection etc. system can fetch data from there.
