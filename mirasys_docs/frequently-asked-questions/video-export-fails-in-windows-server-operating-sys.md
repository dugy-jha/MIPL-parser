Video Export fails in Windows Server operating system
Problem

Windows Server is used. When trying to export video with MediaExporter or Spotter Player, error is seen: "Failed to load media writer". 

Solution

Windows Server machines you need to install Windows feature "Desktop Experience" and also "Media Foundation". It is needed to export videos.
