IP camera is found, but there is “No signal” from the camera

In cases where the IP camera is found, but there is “No signal” from the camera, please go through the following checklist:

Have you set the camera time and timezone the same as Mirasys VMS?

Has the latest firmware been installed on the camera?

Has the latest camera driver been installed to the VMS?

Are the following ports opened between VMS and the camera?

RTSP: 554

UDP: 3556-4556
