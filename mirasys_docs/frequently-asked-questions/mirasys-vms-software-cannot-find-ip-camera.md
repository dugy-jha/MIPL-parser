Mirasys VMS software cannot find IP camera

In cases where Mirasys VMS software cannot find IP camera, please go through the following check-list:

Have you set the camera time and timezone the same as Mirasys VMS?

Has the latest firmware been installed on the camera?

Has the latest camera driver been installed to the VMS?

Are ARP requests allowed in the network?

Is the camera physically connected to the network?

Is the camera connection intact?

Is the camera receiving power / is it switched on?
