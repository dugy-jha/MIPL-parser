Mehrere Kameras auf dem Geräte-Tab
Ein Einzelkamera-Vollbildmodus

Eine einzelne Kamera kann maximiert werden, um den gesamten Arbeitsbereich abzudecken, indem Sie mit der Maus LINKS doppelklicken.Der zweite Mausklick LINKS doppeltklickt eine einzelne Kamera auf die ursprüngliche Größe zurück.

Geräteregisterkarte im Vollbildmodus

Eine ganze Geräteregisterkarte kann maximiert werden, um den gesamten Arbeitsbereich abzudecken, indem Sie mit der Maus RECHTS doppelklicken
