Bearbeiten des Layouts

Wählen Sie die benötigten Geräte im Spotter-Arbeitsbereich aus

Gehe zu den Layouts

Wählen Sie Layouts bearbeiten




Wählen Sie das Layout aus der Liste aus

Ändern Sie den Namen, falls erforderlich

Legen Sie bei Bedarf Zeitplaneinstellungen fest

Klicken Sie auf Mit Strom ersetzen

Klicken Sie auf Speichern
