Profile Map Devices (Plugins)

Wenn ein Kunde die Kamera über die Profilkarte öffnet, wird automatisch die Registerkarte Profilkartengeräte geöffnet und zeigt diese Kameras in Echtzeit an.

Der Standardrastertyp für Profile Map Devices ist in den Spotter-Einstellungen\Plugins\Profile Map Devices. definiert
