Digitale E/A (Echtzeit-modus)

Die Breite des Komponentenfelds kann angepasst werden, und es kann auch reduziert werden, um nur die E/A-Steuerung anzuzeigen. Wenn die E/A-Geräte der Kamerarasterzelle hinzugefügt werden, übernehmen sie die gesamte Rasterzelle.




Ausgänge verfügen über ein kleines Pulldown-Menü, das die Steuerung von Zustandsschalter, Impuls und Impulsdauer ermöglicht.

Bitte beachten Sie, dass die Ausgangszustände auch vom Gerätebaum aus umgeschaltet werden können, indem Sie auf die Ausgangssteuerung klicken.Die Standardaktion der Ausgänge wird im Abschnitt System Manager – Profile definiert.
