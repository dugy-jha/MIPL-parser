Speichern des Layouts

Öffnen Sie alle benötigten Komponenten für den Arbeitsbereich

Klicken Sie auf Layouts

Wählen Sie Neues Layout

Geben Sie den Namen des Layouts ein

Geben Sie die Beschreibung ein, falls erforderlich

Legen Sie bei Bedarf Freigabeoptionen fest

Legen Sie bei Bedarf Zeitplanoptionen fest

Klicken Sie auf Speichern
