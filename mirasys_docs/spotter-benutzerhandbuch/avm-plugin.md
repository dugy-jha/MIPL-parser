AVM-Plugin
AVM-Geräte-Raster

Der Benutzer kann den AVM über das Geräteraster steuern, während er die Kameras im selben Spotter-Fenster sieht. Mehrere Spotter-Fenster und Monitore sind nicht erforderlich.

Um AVM im Device Grid zu verwenden, öffnen Sie das AVM Plugin innerhalb des Device Grids.

Zeitsuche AVM von der Bedienerkonsole aus.

Der Zeitschieberegler kann für die Zeitsuche an der AVM-Bedienerkonsole verwendet werden. Wenn der Benutzer den grünen Ballon des Zeitschiebers loslässt, sendet die Bedienerkonsole eine zu suchende Zeit an den Display Server.
