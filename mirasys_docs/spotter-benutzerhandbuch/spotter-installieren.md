Spotter installieren

Ein Spotter wird mit anderen Anwendungen installiert, wenn das standardmäßige VMS-Installationsprogramm verwendet wird.
 Wenn ein Benutzer nur den Spotter-Client installieren möchte, kann er das dedizierte Nur-Spotter-Installationsprogramm verwenden.
Ein Spotter ist auch als eigenständige ausführbare Datei (SpotterPlayer.exe ), für die keine Installation erforderlich ist.
Diese Version von Spotter kann verwendet werden, um Videoclips, Storyboard-Clips und Videoarchive zu überprüfen.
