Eine einzelne Kamera auf der Geräteregisterkarte

Eine einzelne Kamera kann maximiert werden, um den gesamten Arbeitsbereich abzudecken, indem Sie mit der Maus RECHTS doppelklicken.
Der zweite Mausklick RECHTS doppelklickt stellt die Originalgröße wieder her.
