Kennzeichen hinzufügen

Wählen Sie die Schwarze Liste oder Weiße Liste

Klicken Sie auf Hinzufügen

Geben Sie die Nummer des Kennzeichens ein

Klicken Sie auf Speichern

Kennzeichen aus der Suchansicht hinzufügen

Doppelklicken Sie auf das Kennzeichenfeld

Klicken Sie mit der rechten Maustaste oben auf das Kennzeichen

Klicken Sie auf Kopieren

4. Öffnen Sie Listen

5. Aktuelle Liste auswählen

6. Klicken Sie auf Hinzufügen

7. Plattennummer einfügen

8. Klicken Sie auf Speichern
