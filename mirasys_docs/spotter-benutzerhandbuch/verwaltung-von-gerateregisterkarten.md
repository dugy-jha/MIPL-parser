Verwaltung von Geräteregisterkarten

Nur Streams von Kameras, die sich auf der geöffneten Registerkarte befinden, werden an den Spotter gesendet.Kameras auf den „versteckten“ Registerkarten verbrauchen keine Netzwerkbandbreite für den Spotter, werden aber sofort fortgesetzt, wenn die Registerkarte aktiviert wird.Registerkarten können durch Ziehen neu angeordnet werden an eine andere Position.Sie können auch außerhalb des Spotter-Fensters gezogen werden, um den Inhalt in ein neues, automatisch erstelltes Fenster zu verschieben.Wenn beim Ziehen die Strg-Taste gedrückt wird, wird eine Kopie erstellt, anstatt sie zu verschieben.

Häufig verwendete Tastaturkürzel:

Kurzbefehl

	

Beschreibung




Ctrl+Shift+T

	

Tab-Menü öffnen.




Ctrl+Shift+W

	

Aktuelle Registerkarte schließen.




Ctrl+Tab

	

Wählen Sie die nächste Registerkarte. Wenn die aktuell ausgewählte Registerkarte die letzte ist, wählen Sie die erste Registerkarte im Fenster aus. Die Registerkarte Hinzufügen (+) ist nicht ausgewählt.




Ctrl+Shift+Tab

	

Wählen Sie die vorherige Registerkarte aus. Wenn die aktuell ausgewählte Registerkarte die erste Registerkarte ist, wählen Sie die letzte Registerkarte im Fenster aus. Die Registerkarte Hinzufügen (+) ist nicht ausgewählt.




Ctrl+Alt+V

	

Tab-Steuerelemente aus-/einblenden.
