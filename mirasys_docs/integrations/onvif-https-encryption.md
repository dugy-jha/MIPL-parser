Select Application Type --> Device

Select Profiles --> G, S and T

Select Manufacturer

Select Product Name

Select Feature List

Click CTRL+F

Search text RTPRTSPHTTPS

If you find text RTPRTSPHTTPS, then the camera model support HTTPS stream encryption
