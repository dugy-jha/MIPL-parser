Source: https://documentation.mirasys.com/integrations/integrations/install-mirasys-reporting-grafana

Install Mirasys Reporting Grafana

Download latest package from Mirasys Extranet.

Extract downloaded package

Browse to the Installation folder

Start the installation by clicking the installation package

Click Next

Accept the License Agreement and click Next

Use all default settings and click Next

Click Install

Click Finish

After the installation check that Grafana service is running
