Login Grafana

If the service is running, then open the browser on the master server and enter an address http://127.0.0.1:3000. 

If you run the browser on some other machine, please enter the master server IP address instead of 127.0.0.1

The default username is admin and password admin. It is recommended to change the default password as soon as possible.
