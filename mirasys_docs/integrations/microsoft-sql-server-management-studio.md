Microsoft SQL Server Management Studio

There is also an option that you want to create your own user account in Microsoft SQL which is used for reading database information. 

You may need your own account if you want to example make the connection more secure. 

When you have installed VMS using the Full Installer package this normally use only Windows Authentication. 

This means that you need to enable Mixed Mode in Microsoft SQL which allow using your own account for database login. 

All needed changes and configuration for own user account can be done in Microsoft SQL Server Management Studio. 

This program can be found in Start Menu.
