AVM Glossary
AVM DS = Agile Virtual Matrix Display Server

The monitors are connected to Display servers (max 4 monitors per display server)

Normal Mirasys Spotter installation

AVM OC = Agile Virtual Matrix Operator Console

One or more consoles that control what is displayed on the AVM Display Servers

Normal Mirasys Spotter installation

2 monitor recommended
