Ruudukon asettelu

Spotter Web Mobilen kameraruutuun voi lisätä enintään 16 kameraa. Ruudukossa, jossa on kolme kameraa, näkymä näytetään näytön leveyden mukaisena, ja sen jälkeen se näytetään kahdessa sarakkeessa.

Voit maksimoida yksittäisen kameran näkymän napsauttamalla virran yläosaa (otsikkoa), jossa kameran nimi sijaitsee. Yksittäisen kameran näkymässä vasemmassa yläkulmassa olevalla nuolella voit palauttaa pääruudukon.

Voit valita yhden kameran ruudukkonäkymässä, jotta käyttäjä voi ohjata valittua kameraa (valittuja kameroita) PTZ- ja toistoohjausta varten. Näkymää ympäröivä kehys osoittaa, että kamera on valittu.
