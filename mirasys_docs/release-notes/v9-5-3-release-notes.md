Mirasys VMS V9.5.3 release notes
Release date: 15.02.2023
Fixes 




System Manager

Fixed: Turkey localization is handled correctly for detecting driver names.




Spotter 

Fixed: De-warping library selection issue in the Spotter client caused by a change in camera manufacturer information reception and storing.

Fixed: Changes made to display settings at the OS level are applied to Spotter in real time.

Fixed: Dutch translations.




Spotter Web 

Fixed: User access to profiles in the main server has been corrected.




Gateway 

Fixed: H.265 default decoding issue resolved using the H.265 codec defined for the camera in System Manager camera settings.




Watchdog  

Fixed: HTTP responses are properly disposed of to avoid exhaustion at the OS level.  




Drivers  

Fixed: Axis manages time stamps and connections correctly.
