Mirasys VMS V9.4.1 release notes
Release date: 21.3.2021
Enhancements

Spotter Web performance has been improved

Finnish help files added

Latest VCA Core SDK (version 1.5.8) added

Fixes

Motion detection when using the secondary or tertiary stream for VCA

Multi-channel device license management when license contains fewer channels than the device

VMS server connection management on the master server and clients in big systems when the system is starting up

Event sender leak on the master server when client connections are re-established

Spotter Web playback from old V7 and V8 VMS servers

Two-factor authentication functionality

PTZ tours in Spotter Web

VCA Double Knock, AND, and Per Target rules functionality

Other minor and cosmetic changes
