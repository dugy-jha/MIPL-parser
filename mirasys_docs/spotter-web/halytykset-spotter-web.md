Hälytykset - Spotter Web

Hälytyslista voidaan avata käyttöliittymän oikeassa alakulmasta

Jos aktiivisia hälytyksiä ei ole, kuvake on VIHREÄ

Aktiiviset hälytykset

Jos hälytyksiä on aktiivisena, kuvake on PUNAINEN

Kun hälytyslista on avattu näkyville, tämä näyttää aktiiviset hälytykset PUNAISELLA taustavärille ja päättyneet hälytykset HARMAALLA taustavärillä.

Käyttäjä voi nähdä hälytyksen tiedot napsauttamalla hälytyksen nimen pientä nuolta

Seuraava tiedot näytetään:

Hälytyksen nimi

Aloitus aika

Päättymisaika

Hälytyksen kesto

Prioriteetti

Laukaisin

Hälytyksen toiminnot




Hälytyksen avaaminen

Hälytykset voidaan avata valitsemalla hälytys luettelosta

Kun hälytys avataan, kaikki kamerat, jotka on asetettu hälytystoimintakameroiksi, avautuvat
