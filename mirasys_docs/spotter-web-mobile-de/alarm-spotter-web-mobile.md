Alarm - Spotter Web Mobile

Wenn die Alarmliste geöffnet wird, zeigt die Liste aktive Alarme mit ROTEM Hintergrund und beendete Alarme mit GRAUEM Hintergrund an.

Wenn ein Alarm aktiv ist, ist das Alarmsymbol ROT

Details zum Alarm

Der Benutzer kann die Details des Alarms einsehen, indem er auf den kleinen Pfeil am Ende des Alarmnamens klickt

Die folgenden Details werden angezeigt:

Name des Alarms

Startzeit

Endzeit

Dauer

Priorität

Auslöser

Aktionen

Öffnen eines Alarms

Alarme können durch Auswahl eines Alarms in der Liste geöffnet werden.

Wenn ein Alarm geöffnet wird, werden alle Kameras, die als Alarmaktionskameras festgelegt wurden, geöffnet
