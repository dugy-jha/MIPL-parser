Ansicht - Spotter Web Mobile

Um auf die Einstellungen zuzugreifen, klicken Sie auf das Symbol in der oberen rechten Ecke













Neue Ansicht

Öffnen Sie die benötigten Kameras in der Ansicht.

Ändern der Ansicht

Klicken Sie auf den Namen der Ansicht, um die Ansicht zu ändern.

Löschen der Ansicht

Klicken Sie auf das X neben dem Namen der Ansicht, um sie zu löschen, und bestätigen Sie in dem sich öffnenden Dialog, dass Sie die Ansicht löschen möchten, indem Sie auf OK klicken.

Bearbeiten des Ansichtsnamens

Klicken Sie auf den Stift, um den Namen der Ansicht zu ändern.

Ändern des Ansichtsnamens

Klicken Sie auf Speichern
