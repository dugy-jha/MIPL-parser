Profile Map Devices (Plugin)

Kun asiakas avaa kameran profiilikartalta, Profiilikartan laitteet -välilehti avautuu automaattisesti ja näyttää kyseiset kamerat reaaliajassa.

Profiilikarttalaitteiden oletusruudukkotyyppi on määritetty kohdassa Tiedosto\Asetukset\Plugins\Profile Map Devices.
