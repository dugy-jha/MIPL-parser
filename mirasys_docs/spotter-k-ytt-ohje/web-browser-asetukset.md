Web Browser (Asetukset)

Web Browser sisältää seuraavat asetukset:

Se asettaa kotisivun, jonne laajennus menee, kun se avataan.

Ohjainten käyttöoikeudet ja näkyvyys, joilla voidaan hallita, voiko käyttäjä navigoida, päivittää, palata kotisivulle ja muokata URL-osoitetta.

Se valitsee käytetyn Internet Explorerin version.
