Zwei-Faktoren-Authentifizierung (2FA)

Die Zwei-Faktor-Authentifizierung (2FA) bietet eine zusätzliche Sicherheitsebene für Ihr Konto, indem zusätzlich zu Ihrem Benutzernamen und Kennwort ein zweiter Verifizierungsschritt erforderlich ist. Dadurch wird sichergestellt, dass nur autorisierte Benutzer auf das System zugreifen können, wodurch sensible Videodaten und Systemkonfigurationen geschützt werden.

2FA wird in System Manager von Ihrem Systemadministrator aktiviert.

So verwenden Sie 2FA bei der Anmeldung

Melden Sie sich wie gewohnt an, indem Sie Ihren Benutzernamen und Ihr Passwort eingeben.

Sie müssen den QR-Code scannen und den generierten Code eingeben, wenn Sie sich zum ersten Mal anmelden.

Sie werden aufgefordert, den Code aus Ihrer Authenticator-App einzugeben.

Nachdem Sie Ihre Identität mit dem 2FA-Code verifiziert haben, sind Sie sicher beim VMS angemeldet.
