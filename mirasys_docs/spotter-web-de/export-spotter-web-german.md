Spotter Web Export
Materialien exportieren

Klicken Sie auf das Symbol Material exportieren in der unteren rechten Ecke

Legen Sie den Namen für die Datei Export fest

Wählen Sie die zu exportierenden Elemente aus

Startzeit einstellen

Endzeit einstellen

Format auswählen (ASF, MP4 oder SEF)

Klicken Sie auf Start

Downloads

In den Downloads können Benutzer den Fortschritt des Materialexports sehen

Nach dem Export kann der Benutzer das Paket herunterladen

Das heruntergeladene Paket ist gezippt und muss entpackt werden

Downloads löschen

Wählen Sie einen Clip aus der Liste aus

Klicken Sie auf Export löschen
