Alarm

Die Alarmliste kann von der unteren linken Ecke aus geöffnet werden

Wenn es keine aktiven Alarme gibt, ist das Symbol GRÜN

Aktive Alarme

Wenn es aktive Alarme gibt, ist das Symbol ROT

Wenn die Alarmliste geöffnet wird, zeigt die Liste aktive Alarme mit ROT Hintergrund und beendete Alarme mit GRAU Hintergrund

Der Benutzer kann die Details des Alarms anzeigen, indem er auf den kleinen Pfeil am Ende des Alarmnamens klickt

Folgende Details werden angezeigt:

Alarmname

Startzeit

Endzeit

Dauer

Priorität

Abzug

Aktionen

Einen Alarm öffnen

Alarme können durch Auswahl eines Alarms aus der Liste geöffnet werden

Wenn ein Alarm geöffnet wird, werden alle Kameras, die als Alarm-Action-Kameras eingestellt wurden, geöffnet
