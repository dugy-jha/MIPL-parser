VCA - Miten sääntöjä testataan

Avaa Burnt-in Annotaatio

Ota Event Log (tapahtumaloki) käyttöön

Ota Display Line Counters (Näytön linjalaskurit) käyttöön

Ota Display Blob Map (Näytä Blob-kartta) käyttöön

Ota Display Objects (objektien näyttäminen) käyttöön

Ota Object Area (objekti alue) käyttöön

Ota Object Class (objekti Luokka) käyttöön

Avaa oikea kamera ja tarkista, että käynnistetyt tapahtumat näkyvät.
