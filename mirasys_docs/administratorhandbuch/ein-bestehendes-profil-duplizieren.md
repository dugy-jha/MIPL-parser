Ein bestehendes Profil duplizieren

Sie können ein bestehendes Profil in System Manager kopieren, um die Erstellung neuer Profile auf der Grundlage bestehender Profile zu erleichtern.

Das kopierte Profil enthält identische Geräte-, Alarm- und Karteneinstellungen.

Bitte beachten Sie, dass allgemeine Einstellungen wie Beschreibung und Benutzergruppen nicht übernommen werden.

Das neue Profil ist standardmäßig aktiv und kann bearbeitet und gelöscht werden.

Nur Systemadministratoren mit der Fähigkeit „Profil hinzufügen können“ in der Systemmanagerrolle können das Profil kopieren.

Wählen Sie ein Profil in System Manager > Profile.

Klicken Sie auf die Schaltfläche „Profil kopieren“, die sich zwischen „Profil hinzufügen“ und „Profil entfernen“ befindet.


Bitte beachten Sie, dass diese Schaltfläche nur aktiv ist, wenn ein Profil ausgewählt ist.

Wenn Sie auf „Profil kopieren“ klicken, wird ein neues Profil erstellt, dessen Geräte-, Alarm- und Karteneinstellungen mit denen des kopierten Profils identisch sind.

Die neue Profilkopie steht unter dem kopierten Profil mit einer Nummer hinter dem Namen: Wenn der Profilname „Service“ lautet, wird die Kopie „Service 1“ sein. Wenn es bereits ein Profil „Service 1“ gibt, heißt es „Service 2“ usw.

In den allgemeinen Einstellungen kann der Name bearbeitet werden, das Profil kann aktiviert oder deaktiviert werden und eine Beschreibung kann hinzugefügt werden. Die neue Profilkopie ist standardmäßig aktiv.

Klicken Sie auf ✓ zum Speichern. Das Profil ist nun in SM > Benutzergruppen sichtbar und kann zu Benutzergruppen hinzugefügt werden.
