Email Einstellungen

Sie können E-Mail-Adressen und Gruppen angeben, die definiert werden können, um Berichte über Ereignisse zu erhalten, die im Software Watchdog angegeben sind.

So legen Sie die E-Mail-Benachrichtigungseinstellungen fest:

Öffnen Sie auf der Registerkarte System die E-Mail-Einstellungen

Geben Sie die E-Mail-Adresse des Absenders in das Feld Absender ein. Beachten Sie, dass einige E-Mail-Anwendungen so konfiguriert sind, dass sie nur Nachrichten von gültigen E-Mail-Adressen akzeptieren.

Geben Sie den Namen des Postausgangsservers in das Feld Postausgang (SMTP) ein. Der angegebene Server wird zum Senden aller E-Mail-Benachrichtigungen verwendet

Geben Sie die Anmeldeinformationen und den Port für den SMTP-Server in die entsprechenden Felder ein.

Legen Sie die Ereignisse fest, für die Benachrichtigungen gemäß den Anweisungen im Software Watchdog gesendet werden

Notiz: E-Mails werden nicht an alle System-E-Mail-Empfänger gesendet.

Der Administrator kann steuern, welche Watchdog-Ereignisse und -Alarme an welche E-Mail-Empfänger oder Gruppen die E-Mail gesendet wird.

So fügen Sie dem System neue E-Mail-Adressen hinzu:

Öffnen Sie auf der Registerkarte System die E-Mail-Einstellungen

Klicken Sie auf Neue E-Mail-Adresse hinzufügen 

um eine neue Adresse hinzuzufügen.

Geben Sie den Namen und die E-Mail-Adresse des Empfängers in die Felder Name und Adresse ein.

OK klicken.

So fügen Sie dem System eine neue E-Mail-Gruppe hinzu

Öffnen Sie auf der Registerkarte System die E-Mail-Einstellungen

Klicken Sie auf Neue E-Mail-Adresse hinzufügen 

um eine neue Adresse hinzuzufügen.

Geben Sie den Gruppennamen ein

OK klicken

So fügen Sie einer Gruppe einen oder mehrere Empfänger hinzu:

Markieren Sie die gewünschte Gruppe in der Gruppenliste

Markieren Sie den oder die gewünschten Empfänger in der Empfängerliste

Klicken Sie auf den Pfeil 

um die ausgewählten Empfänger der ausgewählten Gruppe hinzuzufügen

Andere verfügbare Aktionen:

Das Bearbeiten von E-Mail-Namen, Adressen, Gruppennamen und das Entfernen von Personen aus Gruppen ist mit den Schaltflächen Bearbeiten 

möglich.

Personen können mit dem Pfeil aus Gruppen entfernt werden. 

Personen und Gruppen können mit der Schaltfläche 

entfernt werden.
Test-E-Mails können mit Schaltfläche 

a an a gesendet werden ausgewählte E-Mail-Adresse mit den im Dialogfeld E-Mail-Einstellungen definierten Einstellungen.
