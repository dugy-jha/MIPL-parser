Aufbau eines Failover-Systems
Starting point

Der Master-Server und 1 Aufzeichnungs-Slave-Server wurden dem System hinzugefügt

Aktivieren des VMS-Failover-Servers zum Aufzeichnungsserver

Öffnen Sie die Registerkarte VMS-Server

Wählen Sie den Slave-Server aus der Liste aus

Klicken Sie auf Allgemein

Definieren Sie VMS-Servergruppen-ID (Standard 1)

Aktivieren VMS-Server-Failover ist für diesen VMS-Server aktiviert

Aktivieren Sie Erkennen von VMS-Serverfehlern bei Verbindungsverlust

Definieren Sie den Wert Verbindung wird nicht aufgebaut nach

Klicken OK

FAILOVER-Server hinzufügen

Öffnen Sie die Registerkarte VMS-Server

Klicken Sie auf VMS-Server hinzufügen

Legen Sie den Namen für den Server fest

Stellen Sie die IP-Adresse des Servers ein

Definieren Sie die VMS-Servergruppen-ID

Aktivieren  Als Failover-VMS-Server verwenden

Click OK

Nach dem Hinzufügen zeigt die Registerkarte VMS-Server die Zeile Failover VMS Server  an
