Hinzufügen, Bearbeiten und Entfernen von Audiogeräten

Das System unterstützt drei grundlegende Arten von Audiokomponenten: unidirektionale analoge und IP-Audiokanäle, bidirektionale IP-Audiokanäle und einen einzelnen Audiokommunikationskanal.

So konfigurieren Sie Audiogeräte:

Öffnen Sie die VMS-Server tab.

Wählen Sie den richtigen Server aus und öffnen Sie die Seite Hardware aus dem Menü. 

Öffnen Sie die Registerkarte Audio.

Wählen Sie die Hinzufügen-Option

Wählen Sie den Capture-Treiber aus der Liste aus.

Wählen Sie eine dieser Optionen:

Mono Wählen Sie diese Option, um zwei Monokanäle zu verwenden.

Stereo Wählen Sie diese Option, um zwei Monokanäle zu einem Stereokanal zu kombinieren.

Klicken Sie auf OK.

Notiz IP-kamerabasierte IP-Audioeingangs- und -ausgangskanäle werden dem System hauptsächlich über die automatischen Kamerasuchtools hinzugefügt.Falls ein IP-kamerabasierter Audiokanal nicht über die Kamerasuchtools hinzugefügt werden kann oder wenn der Kanal verspätet hinzugefügt wird, folgen Sie den Anweisungen Befolgen Sie die Anweisungen oben, um den Audiokanal hinzuzufügen.

So bearbeiten Sie ein Audiogerät:

Öffnen Sie die VMS-Server tab.

Wählen Sie den richtigen Server aus und öffnen Sie die Seite Hardware aus dem Menü. 

Öffnen Sie die Registerkarte Audio.

Wählen Sie den Audiokanal aus.

Click Edit Audio Channel

in the lower right corner of the tab. Das DialogfeldAudio konfigurieren  wird angezeigt.

Bearbeiten Sie die Informationsfelder.

Klicken Sie auf OK.

So entfernen Sie ein Audiogerät:

Öffnen Sie die VMS-Server tab.

Wählen Sie den richtigen Server aus und öffnen Sie die Seite Hardware aus dem Menü. 

Öffnen Sie die Registerkarte Audio.

Wählen Sie den Audiokanal aus.

Klicken Sie auf Letzten Audiokanal aus der Liste entfernen

in der unteren rechten Ecke der Registerkarte.
Hinweis:Sie können ein Audiogerät nicht aus der Mitte der Liste entfernen; nur das zuletzt hinzugefügte Audiogerät kann entfernt werden.

Das letzte Audiogerät in der Liste wird vom Server entfernt.
