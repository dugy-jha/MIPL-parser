VMS-Serveradressen ändern

Wenn sich die IP-Adresse oder der DNS-Name eines Servers ändert, können Sie die neue Adresse / den neuen Namen über das Tool VMS-Serveradressen ändern definieren.




So ändern Sie die IP-Adresse oder den DNS-Namen eines Servers:

Öffnen Sie auf der Registerkarte System die Option VMS-Serveradressen ändern

Klicken Sie auf den Namen des Servers mit der geänderten IP-Adresse.

Klicken Sie auf VMS-Serveradresse ändern.

Geben Sie die neue IP-Adresse oder den DNS-Namen des Servers in das Feld Neue VMS-Serveradresse ein.

OK klicken
