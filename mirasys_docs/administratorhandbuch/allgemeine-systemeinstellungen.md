Systemsprache

Passwortverwendung
Sie können das System so konfigurieren, dass für alle Benutzer zwei separate Passwörter erforderlich sind.
Aktivieren Sie dazu die Option „Zweites Passwort verwenden“ in den allgemeinen Systemeinstellungen.
In diesem Modus müssen alle Benutzer zwei Passwörter angeben.
Das standardmäßige zweite Passwort ist leer. Diese Funktion ermöglicht die Einschränkung, sodass keine einzelne Person Videos allein überprüfen kann.
Wenn ein Passwort einer Person und das andere Passwort einer anderen Person bekannt ist, müssen beide Personen bei der Videoüberprüfung anwesend sein.
