ONVIF Profile M

Unser System unterstützt jetzt das ONVIF-Profil M. Mit diesem wichtigen Update kann unser VMS effektiv auf Alarmauslöser unterstützender Kameras reagieren und so die Sicherheit und Reaktionsfähigkeit Ihrer Überwachungseinrichtung verbessern.

Um eine nahtlose Integration und Konformität zu gewährleisten, haben wir diese Funktion mit führenden Kameramarken, darunter Axis, Bosch und Hanwha, eingehend getestet.

ONVIF Alarm

Sobald Sie ein Gerät hinzugefügt haben, das das ONVIF-Profil M unterstützt, müssen Sie keine besonderen Schritte unternehmen, um diese Auslöser zu aktivieren oder zu deaktivieren. Sie werden automatisch zu den Metadatenauslösern der Kamera hinzugefügt. In der Alarmkonfiguration im System-Manager wird dies auf der Registerkarte Auslöser angezeigt und als ONVIF aufgeführt. Sie können diese zum Erstellen neuer Alarme verwenden.

Einen ONVIF-Alarm auslösen

Die Gerätealarme/-auslöser selbst sollten über die Webschnittstelle des Geräts konfiguriert werden, da der System Manager keine Benutzeroberfläche für ihre Konfiguration hat. Diese Konfiguration sollte durchgeführt werden, bevor Sie das Gerät zu VMS hinzufügen, oder die Gerätefunktionen sollten im System Manager aktualisiert werden, um die Änderungen am Gerät zu verstehen.

System Manager starten.

Fügen Sie ein Gerät mit ONVIF Profil M Unterstützung hinzu.

Gehen Sie zur Registerkarte VMS-Server.

Wählen Sie die Menüpunkt Alarm.

Wählen Sie einen Neuen Alarm.

Die Registerkarte Allgemein wird geöffnet, in der Sie dem Alarm einen Namen geben und auswählen können, welche Profile den Alarm verwenden sollen.

Gehen Sie zur Registerkarte Auslöser.

Wählen Sie als Typ Metadaten.

Wählen Sie den Typ der ONVIF-Metadaten, die Sie zur Auslösung des Alarms verwenden möchten.

Gehen Sie auf die Registerkarte Aktionen und wählen Sie die gewünschte Aktion für den Alarm aus.

Gehen Sie auf die Registerkarte Kalender und wählen Sie aus, welche Tage/Stunden für den Alarm aktiviert sind. Standardmäßig ist der Alarm für jeden Tag 24 Stunden lang aktiv.

Klicken Sie abschließend auf OK am unteren Rand.
