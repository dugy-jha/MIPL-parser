Lizenzverwaltung
So importieren Sie einen Lizenzschlüssel:

Klicken Sie auf Lizenz aus Datei importieren

Navigieren Sie zum Speicherort der Lizenzdatei

OK klicken Die neue Lizenz wird sofort aktualisiert.

So exportieren Sie einen Lizenzschlüssel:

Klicken Sie auf Lizenzschlüssel in Datei exportieren, um eine Textdatei für die Lizenz zu erstellen, oder auf Lizenzschlüssel in die Zwischenablage exportieren, um den Schlüssel in die Zwischenablage zu kopieren.

Wenn Sie die Lizenz in eine Datei exportieren, legen Sie den Zielordner und den Namen der Datei fest.

OK klicken.

So exportieren Sie die VCA Core HW GUID

Klicken Sie auf VCA Core HW GUID in die Zwischenablage exportieren

Wählen Sie auch die Lizenzseriennummer exportieren

OK klicken
