Hinzufügen und Entfernen von VMS-Servern

Sie können (je nach Lizenz) 1 bis unbegrenzt viele Server in einem System haben.
Ein Server sollte nicht zu mehr als einem Master Server (SMServer) gehören.
Sie können für jeden Server ein Passwort angeben.
Das System fordert Sie zur Eingabe des Passworts auf, wenn jemand versucht, den Server zu einem anderen System hinzuzufügen.

So fügen Sie dem System einen Server hinzu (Aufzeichnungsserver):

Öffnen Sie die Registerkarte VMS-Server

Klicken Sie auf VMS-Server hinzufügen

Das Dialogfenster Allgemeine Einstellungen  wird angezeigt.

Geben Sie den Namen für den Server ein

Geben Sie bei Bedarf eine Beschreibung ein

Geben Sie die IP-Adresse oder den DNS-Namen des Servers ein.

Geben Sie bei Bedarf das Passwort für den Server ein

Klicken OK Der Server und die damit verbundenen Geräte (Kameras und Audiokanäle) werden der Liste hinzugefügt.

Notiz: Wenn der Server passwortgeschützt ist, fordert das System zur Eingabe des Passworts auf.

So entfernen Sie einen Server aus dem System:

Wählen Sie den Server aus, den Sie entfernen möchten.

Click Remove VMS Server 

Klicken Sie zur Bestätigung auf OK.

Verbindungsstatus:

Wenn die Verbindung zum Server unterbrochen wird, versucht die System Manager-Anwendung automatisch, eine Verbindung zum Server herzustellen.

HINWEIS:

Wenn Sie Mirasys VMS als Slave-Server hinzugefügt haben, gehen Sie bitte wie folgt vor:

Ändern Sie das Admin-Benutzerkennwort mit einem Slave-Server System Manager

Deaktivieren Sie den SMServer-Dienst vom Slave-Server
