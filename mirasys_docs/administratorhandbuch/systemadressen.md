Systemadressen

Der Master Server ist der zentrale Server eines Überwachungssystems.

Alle anderen VMS-Server verbinden sich damit und alle Client-Anwendungen kommunizieren über den Master-Server. Während der Anmeldephase können die Clientanwendungen den Masterserver auswählen, zu dem sie eine Verbindung herstellen.




Sie können mehrere Master-Server-Adressen definieren, mit denen sich die Client-Anwendungen verbinden können.
Die Adressen können als IP-Adressen (z. B. http://195.168.0.1) oder DNS-Namen (z. B.  http://www.example.com) bereitgestellt werden.

Notiz: Benutzer können sich mit jeder der definierten Master-Server-Adressen verbinden, vorausgesetzt, sie haben einen kompatiblen Benutzernamen und ein kompatibles Passwort für den Master-Server.

So fügen Sie eine Masterserver-Adresse hinzu

Öffnen Sie auf der Registerkarte System die Systemadressen

Klicken Sie auf Neue Systemadresse hinzufügen 

Geben Sie die neue Systemadresse (entweder IP-Adresse oder DNS-Name) in das Feld Hinzufügen ein.

OK klicken

So bearbeiten Sie eine Master-Server-Adresse:

Öffnen Sie auf der Registerkarte System die Option Systemadressen.

Klicken Sie auf die Master-Server-Adresse mit der geänderten IP-Adresse.

Klicken Sie auf Systemadresse ändern .

Geben Sie die neue IP-Adresse oder den DNS-Namen des DVR in das Feld Systemadresse ändern ein.

OK klicken

So entfernen Sie eine Master-Server-Adresse:

Öffnen Sie auf der Registerkarte System die Systemadressen

Klicken Sie auf die Master-Server-Adresse, die Sie entfernen möchten.

Klicken Sie auf Systemadresse entfernen.
