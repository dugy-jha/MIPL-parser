Käyttäjien roolit

Järjestelmä tukee seuraavan tyyppisiä käyttäjärooleja (määritelty käyttäjäryhmien kautta):

System Manager Enterprise rooli Järjestelmänvalvojat voivat kirjautua sisään System Manageriin ja muuttaa kaikkia asetuksia, kuten muuttaa kameran asetuksia tai lisätä uusia profiileja tai käyttäjätilejä.

Seurantarooli  Käyttäjät, joilla on valvontaoikeudet, voivat kirjautua sisään System Manageriin ja valvoa järjestelmää Järjestelmä-välilehdellä, mutta he eivät saa muuttaa asetuksia.

Gateway rooli: jos tämä rooli on aktiivinen, käyttäjäryhmä voi käyttää VMS-yhdyskäytävää 

Mirasys Spotter Enterprise rooli:  Loppukäyttäjät voivat kirjautua sisään Spotteriin, mutta eivät System Manageriin.

Spotter Web rooli: Loppukäyttäjät voivat kirjautua sisään Spotter Webiin
